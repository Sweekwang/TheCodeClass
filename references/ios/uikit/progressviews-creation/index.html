How to create Progress View in iOS

Progress View - iOS Swift

Download ResourcesOfficial Reference

UIProgressView helps to track the progress of your task. When the progress view is filed up, its means that the task has completed. Progress view give user the feedback that the app is working and also indicate when the task is going to complete.

Take note. If the progress view fill up slowly, user might think that your app is frozen 😢

How to create a Progress View?

Step 1: In Main.storyboard file, click on the + icon to open the Object Library window and drag and drop a progess view on the view.

Step 2: Click on the Attributes Inspector icon or go to Choose View ➤ Inspectors ➤ Show Attributes Inspector. You will be able to see attributes available for a progress view. Change accordingly to the properties you want.

AttributesMeaning
ProgessProgess is the value between 0 to 1. 0 is the begining of the progess while 1 is the completion of the progress

Step 3:Connect your progess view to ViewController.swift file. (Click on the progess view, hold down the control key, and drag from the progess view to the ViewController.swift file relase and a popup window appears.)

Choose the following parameters: connection equals IBOutlet, type equals UIProgressView and give a name to it.

Swift Code:

@IBOutlet weak var progessViewName: UIProgressView!

You can update the progess by changing the progess view's progess

Swift Code:

progessViewName.progress = 0.8

Another way to update the progess value. This can create a smoother looks.

Swift Code:

progessViewName.setProgress(0.8, animated: true)

Example:

Below is an example, when the progess is updated using a countdown timer. When the time is in 60s, the progess is completed.

Swift Code:

@IBOutlet weak var progessViewName: UIProgressView!
var counter: Double = 1

override func viewDidLoad() {
    super.viewDidLoad()

    Timer.scheduledTimer(timeInterval: 1.0 target: self, selector: #selector(updateCounter), userInfo: nil, repeats: true)

}


@objc func updateCounter() {
    if counter <= 60 {
    progessViewName.progress = Float ( counter / 60 )
    counter += 1
    }
}
