How to create gradient effects?

CAGradientLayer

Official Reference

Creating a gradient layer to fill your shape and etc.

Step 1: Create a CAGradientLayer and this layer create a color gradient containing a number of colors.

Swift Code:

let gradientLayer = CAGradientLayer()

Step 2: Add a list of colors into the CAGradientLayer. (In order to see your gradient, you can add it into your UIView or view.)

In the example, below, the gradient is add into the UIView and the following gradient is shown:

Swift Code:

gradientLayer.colors = [ UIColor.red.cgColor, UIColor.yellow.cgColor, UIColor.green.cgColor]
gradientLayer.frame = viewName.bounds
viewName.layer.addSublayer(gradientLayer)

Step 3: Changing the startPoint and endPoint able to create different gradient style. The top right hand corner has a postion of (0.0, 0.0) while the bottom right is (1.0, 1.0) as show in the example below . The Default value is (0.5,0.0) for startPoint and (0.5, 1.0) for endPoint.

Change the postion is able to create different linear effect like the example below: The start point of is [0.0, 0.0] and the end point is [1.0, 1.0].

Swift Code:

gradientLayer.startPoint = CGPoint(x: 0.0, y: 0.0)
gradientLayer.startPoint = CGPoint(x: 1.0, y: 1.0)

Step 4: Changing the type of the gradient. Three type of gradient type is available axial (default), conic and radial

typeMeaning
axial

Swift Code:

gradientLayer.type = .axial

conic

The startPoint is (0.5, 0.5) and endPoint is (1.0, 1.0)

Swift Code:

gradientLayer.type = .conic

radial

The startPoint is (0.5, 0.5) and endPoint is (1.0, 1.0)

Swift Code:

gradientLayer.type = .radial

Step 5: Changing the locations of the gradient able to let you choose the amount of color to shown in the gradient. By default, the colors are spread uniformly across the layer.

Using axial type as an example with a startPoint of (0.0, 0.5) and endPoint of (1.0, 0.5). The default is spread uniformly from 0.0 to 1.0 as shown below and you can change the position to create more red by changing the yellow color postion to 0.75.

Swift Code:

gradientLayer.locations = [ 0.0, 0.75, 1.0]
