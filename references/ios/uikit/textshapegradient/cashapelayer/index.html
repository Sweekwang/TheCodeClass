How to create shape with CAShapeLayer and UIBezierPath?

CAShapeLayer and UIBezierPath

Official Reference

You can draw shape with CAShapeLayer which is a CALayer sub-class that allows you to draw shapes. In this class, we will be using UIBezierPath to define the shape and after the shape is created, we can change properties such as fileColor, strokeColor and more!

Step 1: Create a CAShapeLayer Object.

Swift Code:

let shapeLayer = CAShapeLayer()

Step 2: Create a UIBezierPath Object. This will create shape for your CAShapeLayer.

ovalIn - is used to create full circle. You have to state the rect: The rectangle in which to inscribe an oval. Example:

Swift Code:

let circularPath = UIBezierPath(ovalIn: CGRect(x: 100, y: 100, width: 200, height: 200))

arcCenter - is used to create non full circle. You have to state center: the center position of the arc, radius of the arc, startAngle and endAngleof the arc in radians and clockwise: the direction to draw the arc.

Swift Code:

let circularPath = UIBezierPath(arcCenter: view.center, radius: 100, startAngle: 0, endAngle: CGFloat.pi, clockwise: true)

roundedRect - is used to create rectangle with corner radius.You have to state the roundedRect: the basic shape of the path in CGRect such as its position, height and width and cornerRadius: the radius of each corner oval.

Swift Code:

let circularPath = UIBezierPath(roundedRect: CGRect(x: 100, y: 200, width: 200, height: 200), cornerRadius: 20)

Step 3: Add the path into the CAShapeLayer Object and then finally, add it into your view.

Swift Code:

shapeLayer.path = circularPath.cgPath
view.layer.addSublayer(shapeLayer)

Optional: You can change the properties of the shape.

AttributesMeaning
fillColorChanging the color of the fill or area. (default: black)

Swift Code:

shapeLayer.fillColor = UIColor.blue.cgColor

lineWidthThe size of the border. (To see the border you have to set strokeColor and strokeEnd)

Swift Code:

shapeLayer.lineWidth = 10

strokeColorChanging the color of thr border.

Swift Code:

shapeLayer.strokeColor = UIColor.red.cgColor
