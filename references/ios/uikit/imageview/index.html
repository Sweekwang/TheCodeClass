How to create an Image View in iOS

Image View - iOS Swift

Download ResourcesOfficial Reference

Photo by Tyler Lastovich on Unsplash

Not all data is text. Image View allows you to display graphics for decorative purposes or provide information for the user. You can display image such as .jpg or .png images. However, image view also allow interaction from the user.

How to add Image to Xcode?

To add an image to Xcode, go to Assets.xcassets and drag and drop a .jpg or .png image from the Finder window to the image into Assets.xcassets folder.


1x, 2x, 3x meaning (Scale Factor)

iOS uses point as an measurement to place content on screen which map it to pixels in display.1x means that one pixal is equal to one point. For example, 1x image that has a resolution of 100px × 100px will be 200px × 200px in 2x and 300px × 300px in 3x. Depending on the device the have different scale factor.

How to create an Image View?

Step 1: In Main.storyboard file, click on the + icon to open the Object Library window and drag and drop a image view on the view.

Step 2: Choose the image you want to use for the image view by changing the image name in the Attributes panel. (Click on the Attributes Inspector icon or go to Choose View ➤ Inspectors ➤ Show Attributes Inspector.)

Content Mode (How Images Are Scaled)

ContentMode property specify how the image is display in the UIImageview. If the size of the image changes, it will scale accordingly to the content mode you stated.

AttributesMeaning
Aspect FitStretch the image to fill up the width and height of the image view.
Scale to FillScales the image until the biggest side reach the maximum height or width.
Aspect FillScales the image until the smallest side reach the maximum height or width.
RedrawStretch the image to fill up the width and height of the image view. Similair to aspect fit. However, there is a difference on how they are draw.
Center | Top | Bottom | Left | RightDisplays the image center, Top, Bottom, Left or Righ within the image view. If the image is larger than the image view, then the rest of the image appears outside of the image view.
Top Left | Top RightDisplays the top left or top right of the image within the image view. If the image is larger than the image view, then the rest of the image appears outside of the image view.
Bottom Left | Bottom RightDisplays the bottom left or bottom right of the image within the image view. If the image is larger than the image view, then the rest of the image appears outside of the image view.

Step 3: Changing the attribute with code. Connect your progess view to ViewController.swift file. (Click on the progess view, hold down the control key, and drag from the progess view to the ViewController.swift file relase and a popup window appears.)

Choose the following parameters: connection equals IBOutlet, type equals UIImageView and give a name to it.

Swift Code:

@IBOutlet weak var imageviewName: UIImageView!

Chnaging the attribute

Swift Code:

imageviewName.image = #imageLiteral(resourceName: "imagename")
imageviewName.contentMode = .scaleAspectFit

Step 4: Interactive image view.

By default, an image view all touch interaction and therefore, you have to enable the it by either checking the User Interaction Enabled check box or using code.



Swift Code:

imageviewName.isUserInteractionEnabled = true

Responding to Touch Events : There are two ways to handle the touch events. One is by attaching gesture recognizers or using ther event handling techniques

Event Handling:
Type the following method and write a if statement to detect where the user touches.

Swift Code:

override func touchesBegan( _ touches: Set <UITouch> , with event: UIEvent?){
    let touch = touches.first
    if touch?.view == imageviewName {
        print("Touched Image")
     }
}

Gesture recognizers:
1) Click on the + icon to open the Object Library window and drag and drop a an gesture recognizers such as Tap Gesture Recognizer on the image view.

2) Click on the gesture recognizers, hold down the Control key, and drag from the gesture recognizers to the ViewController.swift file release and a popup window appears.

Choose the following parameters: connection equals Action, type equals UITapGestureRecognizer and give a name to it.

Swift Code:

@IBAction func tappedOnImage( _ sender: UITapGestureRecognizer){
    print("Touched Image")
}
