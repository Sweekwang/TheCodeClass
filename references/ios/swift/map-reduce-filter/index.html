What is Map Reduce Filter in Swift?

Map Reduce Filter - Swift

Iterating over collections. These functions come in handy as you can apply to a collection of items to perform a certain operation and it return the collection of item back rather then writing a loop and then append it into a new list like that one stated below.

Swift Code:

var arrayOfItems = [1,4,4,10,3,8]
var arraryX2 = [Int]( )

for i in arrayOfItems {
    arraryX2.append( i * 2 )
}

1) Map

Rather then writing a loop like the one shown above, you can use a map method to apply the same concept without writing a loop.

Swift Code:

var arrayOfItems = [1,4,4,10,3,8]
var arraryX2 = arrayOfItems.map({(numberOne) -> Int in
     numberOne * 2
})

print( arraryX2 )

2) Compact Map

Compact back work the same way as map. Just that take in the value as an optional and return only non-nil value.

Swift Code:

var arrayOfItems = [ "1" , "4" , "s" , "10" , "3" , "8" ]
var arrary2Int = arrayOfItems.compactMap({numberOne in
     Int(numberOne)
})

print( arrary2Int )


Output:

[1, 4, 10, 3, 8]

3) Filter

Similar to map, it will go loop through element by element and only return those that are True

Swift Code:

var arrayOfItems = [1,4,4,10,3,8]
var arraryX2 = arrayOfItems.filter({(numberOne) -> Int in
     numberOne > 5
})

print( arraryX2 )


Output:

[10 , 8]

4) First

Same as filter, but if you only interested in the first element.

Swift Code:

var arrayOfItems = [1,4,4,10,3,8]
var first = arrayOfItems.first{(numberOne) -> Int in
     numberOne > 5
}

print( first )


Output:

Optional(10)

5) Reduce

Reduce takes in a starting value and a closure and then the closure takes two value, the current value and an element from the array. The closure return the next value which is used as a parameter for the next element of the array.

The example below sum up all the value with a starting value of zero.

Swift Code:

var arrayOfItems = [1,4,4,10,3,8]
var sum = arrayOfItems.reduce(0, {(numberOne, numberTwo) -> in
     numberOne + numberTwo > 5
}

print( sum )


Output:

30
